About Page

What is this?

Finding specific General Education classes that fulfill the specific requirements necessary may be hard. This website is designed to make the process easier by allowing students to choose the categories and search for a list of classes. It also allows students to see how many course options are provided for a specific general education requirement.

How to Use It

The Home page is a search page to find a list of classes that fulfill the selected GenEd category. All that needs to be done is select the categories and press Search. There is nothing that needs to be done on the compare page.

What is being displayed?

The Home Page is displaying a list of classes being offered based on the categories selected by the user. The compare page consists of a histogram that will display the number of classes offered for each general education requirment category.
